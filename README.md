# Basic JS Movement Script

Meant for **400 × 400** pixel movement. Developed from **Code.org Lesson 16**.

## Overview
This repository contains a simple JavaScript movement script intended for projects that use a **400×400** coordinate/canvas space (for example, classroom game projects or Code.org-style sketches).

## Features
- Movement logic designed around a 400×400 play area
- Intended as a learning-friendly starting point for basic keyboard-controlled movement

## Getting Started
1. Copy the code from the move.js.
2. Open the project in Code.org.
3. Run it!

## Customization
Common things you may want to tweak:
- **Canvas/World size**: update any boundary checks if you change from 400×400
- **Speed**: adjust the per-frame/per-keypress step value
- **Controls**: remap key bindings to your preference

This is meant as a base for basic movement.
